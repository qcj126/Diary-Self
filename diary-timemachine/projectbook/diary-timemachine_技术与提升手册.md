# diary-timemachine 模块技术与提升手册

## 一、模块概述

diary-timemachine 是Diary-Self项目的时光机服务模块，提供时间卡片和分类的增删改查功能。

## 二、技术栈

### 2.1 核心技术
- Java 21
- Spring Boot 3.5.14
- Spring Boot Starter Web
- Spring Boot Starter Data Redis
- MyBatis-Plus Spring Boot3 Starter 3.5.5
- MySQL Connector J 9.6.0
- Lombok 1.18.44

### 2.2 服务端口
- **端口**: 8804

## 三、核心功能

### 3.1 分类管理
- 新增分类
- 删除分类
- 修改分类
- 查询分类

### 3.2 卡片管理
- 新增卡片
- 删除卡片
- 修改卡片
- 分页查询卡片

### 3.3 功能特点
- 支持图片关联
- 支持分类管理
- 支持时间记录
- 支持逻辑删除

## 四、优化建议

### 4.1 代码优化

## 五、程序风险

### 5.1 中风险
1. 大数据量查询慢
2. 缺少权限控制

### 5.2 低风险
1. 日期类型混用（Date和LocalDateTime）
2. 缺少数据校验
