# diary-recipe 模块技术与提升手册

## 一、模块概述

diary-recipe 是Diary-Self项目的食谱服务模块，提供食谱的增删改查和分页查询功能。

## 二、技术栈

### 2.1 核心技术
- Java 21
- Spring Boot 3.5.14
- Spring Boot Starter Web
- Spring Boot Starter Data Redis
- MyBatis Spring Boot Starter
- Druid 1.2.28
- MySQL Connector J 9.6.0
- Lombok 1.18.44

### 2.2 服务端口
- **端口**: 8803

## 三、核心功能

### 3.1 食谱管理
- 新增食谱（包含食材和步骤）
- 删除食谱
- 修改食谱
- 分页查询食谱

### 3.2 食谱属性
- 标题、封面图、简介
- 分类、餐别、难度
- 烹饪时长、情感故事
- 纪念日专属
- 食材列表
- 步骤列表

### 3.3 查询功能
- 分页查询
- 分类筛选
- 餐别筛选
- 难度筛选
- 关键词搜索

### 5.1 中风险

### 5.2 低风险
