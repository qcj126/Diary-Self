# diary-file 模块技术与提升手册

## 一、模块概述

diary-file 是Diary-Self项目的文件服务模块，提供文件上传、下载、管理等功能，包括：
- 图片上传（批量）
- 视频上传
- 图片URL查询
- 文件下载
- OSS集成
- 异步处理
- 消息队列集成

## 二、技术栈

### 2.1 核心技术
- **Java 21**: 项目基础语言版本
- **Spring Boot 3.5.14**: 基础框架
- **Spring Boot Starter Web**: Web模块支持
- **Spring Boot Starter Data Redis**: Redis支持
- **Spring Boot Starter AMQP**: RabbitMQ支持
- **MyBatis Spring Boot Starter**: MyBatis集成
- **Druid 1.2.28**: 数据库连接池
- **Aliyun SDK OSS 3.17.4**: 阿里云OSS存储
- **MySQL Connector J 9.6.0**: MySQL驱动
- **Lombok 1.18.44**: 简化代码开发

### 2.2 依赖管理

### 2.3 服务端口
- **端口**: 8802

## 三、核心功能

### 3.1 文件上传
- **图片上传**: 支持批量上传图片
- **视频上传**: 支持视频文件上传
- **异步处理**: 上传到OSS采用异步方式
- **消息队列**: 上传成功后发送MQ消息

### 3.2 文件下载
- **批量下载**: 支持批量下载图片
- **OSS签名URL**: 生成临时访问链接
- **本地缓存**: 下载文件到本地

### 3.3 文件查询
- **图片URL查询**: 根据ID查询图片URL
- **批量查询**: 支持批量查询

### 3.4 异步服务
- **异步上传**: 使用@Async异步上传
- **线程池**: 自定义线程池处理
- **消息发送**: 上传成功后发送MQ消息

### 3.5 监控
- **Actuator**: 健康检查和监控端点
- **Prometheus**: 指标收集和导出

## 六、程序风险

### 6.1 高风险
1. **文件存储**
   - OSS凭证泄漏风险
   - 建议使用STS临时凭证
   - 限制Bucket权限

2. **大文件处理**
   - 大文件上传可能导致OOM
   - 建议限制文件大小
   - 增加流式处理

3. **异步任务**
   - 异步任务失败可能丢失
   - 建议增加持久化
   - 增加补偿机制

### 6.2 中风险
1. **消息队列**
   - MQ消息可能丢失
   - 建议开启持久化
   - 增加消息确认

2. **并发问题**
   - 批量上传可能存在并发问题
   - 建议增加分布式锁
   - 增加幂等性处理

### 6.3 低风险
1. **临时文件**
   - 下载产生的临时文件需清理
   - 建议定时清理

2. **日志敏感信息**
   - 日志中可能包含文件路径
   - 建议脱敏处理

## 七、维护建议
